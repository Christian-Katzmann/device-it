# device-it

Turn a local web project into an installable, fullscreen, offline-capable app on
your **phone, tablet, or desktop** — with its own icon and name. The on-device
sibling of `app-it` (which makes macOS Dock apps).

One command inspects the project, builds it, applies a PWA transform to the built
output, deploys to a stable HTTPS URL, and installs it — via QR / Add-to-Home-Screen
on any device, or a zero-touch Apple MDM push once enrolled. It uses whatever host
you already have (Vercel, Netlify, Cloudflare Pages, GitHub Pages) and falls back to
an anonymous, no-account deploy so it works out of the box.

A skill for [Claude Code](https://claude.com/claude-code) and Codex. Start at
[`SKILL.md`](SKILL.md).
